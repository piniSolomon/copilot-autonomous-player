### Learnings
- Copilot CLI can replicate most Claude Code autonomous patterns using background agents as timers
- The session-bound limitation doesn't apply when the user keeps the session alive permanently
- `ask_user` (terminal) + Discord fallback is a viable replacement for `wait_for_reply()`
- `</script>` inside HTML comments within a `<script>` block terminates the block — never use it in comments
- Throttle `renderShop()` to avoid DOM churn that causes Playwright "element detached" errors
